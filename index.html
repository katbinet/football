Estimation — Global Football Players Network (Android + iOS + Web)

Lean UAE-market baseline pricing for an MVP focused on video profiles of football players, simple discovery and contact via phone only, with monthly subscription and automated data deletion.

Scope (MVP)

  • Platforms: Web + Mobile (Android & iOS, single codebase).
  • User roles: Player, Admin/Moderator; sections by gender (Men / Women).
  • Registration (required): name, email, phone, full address (country/region/city...), birth year.
  • Profiles: phone visible; no likes, no comments, no shares.
  • Video: player-only clips, no music policy; basic moderation + violation auto-ban.
  • Discovery flow: Continents → Country flags → Birth year → Player list.
  • Filters: by birth year, country, and gender.
  • Subscription: 1 month access; auto-delete all user data at expiry.
  • Payments: AED pricing; Stripe/Checkout.com with cards + Apple Pay/Google Pay (where available).
  • Language: English (primary). UI kept minimal and fast.

Architecture (lean)

  • Mobile: Flutter for Android/iOS (single codebase, faster delivery).
  • Web: React/Next.js minimal client (same API).
  • Backend: Node.js (Nest/Fastify) or Django; PostgreSQL for relational data.
  • Media: Cloudflare Stream/Mux for upload, transcode, delivery (low OPEX).
  • Auth: email/phone + OTP; rate limits and basic fraud checks.
  • Admin: lightweight moderation queue + audit logs.
  • Compliance: scheduled job to purge accounts exactly on subscription end date.

Subscription Price (User)

USD 10.00 ≈ AED 36.75 (peg 3.675). Recommended in-app price: AED 37 / month (excl. VAT). If VAT applies (5%), user-facing price ≈ AED 38.85.

Gateway fees (typical UAE): ~2.75%–2.9% + AED 1 per charge.

Monthly OPEX (estimate)

  • Hosting + DB: ~AED 250–450/month (starter cluster).
  • Video storage/streaming: usage-based; e.g., 1,000 videos (~30 GB) + 5,000 min streaming ≈ AED 120–220/month.
  • SMS OTP: ~AED 0.15–0.40 per SMS; 1,000 OTPs ≈ AED 150–400.
  • Gateway fees: ~2.75%–2.9% + AED 1 per successful charge.
  • Apple Dev (yearly): ~AED 365; Google Play (one-time): ~AED 92.
  • Domain + email: ~AED 10–40/month (basic).

Actual media costs depend on uploaded minutes, bitrates, and regional traffic; figures are conservative for a lean MVP.

Policies & Compliance

  • Content: player-only videos, no music; violations → auto account deletion.
  • Privacy: delete all user data exactly at subscription end; logs redact PII.
  • Safety: gender separation in listing; report/ban tools for admins.
  • Payments: AED-first pricing; FX handled by gateway if needed.
  • Data export: user can download profile before deletion (optional).
